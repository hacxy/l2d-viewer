# l2d-viewer

一个面向 Live2D 调试场景的可视化工具，用来快速加载模型并查看/操作：

- 模型位置与缩放
- 表情与动作播放
- 参数实时变化与手动调参
- Hit Area 可视化与数据表

本项目基于 React + TypeScript + Vite，依赖本地 `l2d` 库进行模型加载与渲染。

## 功能概览

- **模型加载**：输入 `.model3.json` 或 `.model.json` 地址加载模型
- **状态反馈**：显示 `idle / loading / loaded / error`，加载中有进度遮罩
- **URL 同步**：模型地址写入 `?model=`，支持刷新后自动恢复
- **历史记录**：最近加载地址保存在本地（最多 20 条）
- **变换控制**：X/Y/Scale 滑块调整，支持重置与代码片段复制
- **动作面板**：按分组展示动作，支持播放中点击切换其他动作
- **表情面板**：查看并切换表情，支持随机表情
- **参数面板**：
  - 参数搜索
  - 活跃参数高亮
  - 每参数滑块调值与恢复默认值
  - 虚拟列表优化，降低长参数列表卡顿
- **Hit Area**：
  - 画布覆盖层显示点击区域
  - 底部面板查看区域坐标表格
- **交互操作**：
  - 画布拖拽移动模型
  - 鼠标滚轮缩放模型
- **主题切换**：浅色 / 深色一键切换

## 技术栈

- React 19
- TypeScript
- Vite
- Ant Design 6
- Jotai
- React Router 7

## 环境要求

- Node.js 18+（建议使用较新 LTS）
- pnpm 8+
- 本地存在上游库目录：`../l2d`

> `package.json` 中通过本地依赖引用：`"l2d": "file:../l2d/"`

## 快速开始

```bash
# 1) 安装依赖
pnpm install

# 2) 启动开发环境
pnpm dev
```

默认访问地址通常为：

- [http://localhost:5173](http://localhost:5173)

若端口被占用，Vite 会自动切换到其他端口（例如 5174）。

## 使用说明

1. 在顶部输入模型配置地址（`.model3.json` / `.model.json`）
2. 点击 `Load` 加载模型
3. 通过右侧面板调整位置、缩放并复制 `load()` 代码
4. 在左侧面板切换表情、动作，观察参数实时变化
5. 在底部面板开启 Hit Area 覆盖层并查看坐标数据

## 常见问题

### 1) 安装时报找不到 `../l2d`

请确认 `l2d-viewer` 与 `l2d` 目录是同级关系，例如：

```text
projects/
├── l2d
└── l2d-viewer
```

### 2) 打开参数面板后卡顿

当前已做参数列表虚拟渲染与轮询降频优化。如果模型参数特别多，建议优先使用搜索过滤后再调参。
